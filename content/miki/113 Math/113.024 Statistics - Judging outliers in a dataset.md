# Judging outliers in a dataset

If a number below the first quartile is less than `1.5 * IQR` then it is an outlier. Same for a number greater than the third quartile

- $$Q_1 - (1.5 * IQR)$$
- $$Q_3 + (1.5 * IQR)$$

`IQR` = Interquartile range
